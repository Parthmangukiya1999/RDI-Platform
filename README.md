### RDI Platform ####

# Short Description: #
RDI Platform is a web platform for IIT students to create portfolio websites, share project ideas, set goals, and find team members for real projects.

# Badges # 
(Add later when CI, tests, or versioning are ready)
 - Build Status: Coming soon
 - Test Coverage: Coming soon
 - Version: v0.1 (planned)


# Overview #
Many students have good ideas but no single place to show their skills, manage goals, or find the right people to work with.
RDI Platform solves this by bringing portfolios, projects, learning materials, and collaboration into one system.

# Target users: #
IIT students
Student project teams

# Features #
- Create a personal portfolio website based on academic marking needs
- Share and post project ideas
- Find team members, such as coders or designers, using ratings
- Set personal and project goals
- Access course materials related to projects and goals
- View available projects with duration and status

# Tech Stack #
- Frontend: React
- Backend: Node.js (Express)
- Database: MongoDB (planned)
