##  Projects Included

###  1. Helsinki Population Analysis (Machine Learning Project)

**Objective:**  
Analyze long-term population trends in Helsinki and predict future changes using machine learning.

**Key Features:**
- Data cleaning and preprocessing (Excel → Pandas)
- Trend visualization (Matplotlib, Seaborn)
- Forecasting models:
  - Linear Regression (final model)
  - Polynomial Regression (comparison)
- Clustering (K-Means):
  - High growth
  - Moderate growth
  - Stable/declining areas
- Anomaly detection (Isolation Forest)

**Key Insights:**
- Migration is the main driver of population growth
- Growth is not uniform across districts
- Some areas (e.g., Jätkäsaari) show rapid development
- Population trends are non-linear
